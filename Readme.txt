Se inicio con react-create-app y express para un template basico.
No se incluyen mecanismos de auth ni log ya que no estaban en el pedido, de ser requerido se podrian agregar rapidamente.

Se espera un server MongoDB en 'localhost', port: 27017. SI no existe falla al arranque, si se quiere modificar,
los valores a cambiar estan en el objeto 'config' de app.js

La api nodeJs correra en el puerto 33111, en caso de queres cambiarlo,el valor a cambiar estan en el objeto 'config' de app.js

Para iniciar se puede usar 'npn start' o node 'app.js'
